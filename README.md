# next-boilerplate
Open Source React Next Boilerplate made by TeamClimactic

## Packages
- next
- headlessui
- daisyui
- eslint, prettier
- tailwindcss

## Features
- Basic Boilerplate with all necesities
- Strict Linting Configured
- Completely in TypeScript
- Tailwind, Headless, and DaisyUI configured
